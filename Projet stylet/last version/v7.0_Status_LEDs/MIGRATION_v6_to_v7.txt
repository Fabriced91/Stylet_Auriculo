 🎯 OBJECTIF DE LA MIGRATION
═══════════════════════════════════════════════════════════════════════════════

Remplacement de la LED Nichia 519A (3535 SMD, 350mA) par la LED Nichia NSPW500CS
(5mm traversant, 20mA) pour :

✅ SIMPLIFIER drastiquement le circuit électronique
✅ RÉDUIRE la consommation énergétique (×17.5 moins de courant)
✅ FACILITER l'assemblage (composant traversant vs SMD)
✅ AUGMENTER l'autonomie sur batterie
✅ RÉDUIRE les coûts de production
✅ CONSERVER un CRI >90 pour la qualité thérapeutique

═══ CALCUL RÉSISTANCE SÉRIE (R_LED) ═══

Tension alimentation : VBAT = 3.0V (min) à 4.2V (max)
Tension forward LED  : Vf = 3.0V @ 20mA (typ), max 3.6V

Avec régulateur 3.3V :
  R = (Vsupply - Vf) / If
  R = (3.3V - 3.0V) / 0.020A = 15Ω

Pour limiter courant à 20mA avec batterie pleine (4.2V directement) :
  R = (4.2V - 3.6V) / 0.020A = 30Ω

⚠️ PROBLÈME : Avec 3.0V batterie faible, LED s'éteint (3.0V < Vf 3.0V)

🎯 SOLUTION OPTIMALE : Alimenter LED directement depuis BAT+ (sans LDO)
  + Ajouter résistance 150Ω pour limiter courant

  R = (3.3V - 3.0V) / 0.020A = 150Ω  (valeur standard)
  
  Vérification batterie pleine :
    I_max = (4.2V - 3.0V) / 150Ω = 8mA  ✓ Sécuritaire
  
  Vérification batterie faible :
    I_min = (3.0V - 3.0V) / 150Ω = 0mA  ✗ LED éteinte

🎯 SOLUTION FINALE : Alimenter depuis 3.3V régulé + résistance 47Ω
  
  Mode TRAITEMENT (PWM 100%) :
    I = (3.3V - 3.0V) / 47Ω = 6.4mA  → Augmenter PWM duty cycle
    
  Avec PWM variable, contrôler intensité de 0 à 20mA :
    R = 47Ω (limite hardware à 64mA max si PWM bug → sécurité)
    
  Pour obtenir 20mA :
    Duty_cycle = (3.0V + 20mA × 47Ω) / 3.3V = 0.91 × 3.3V
    → PWM à ~92% pour 20mA

 ✅ AVANTAGES DE LA MIGRATION
═══════════════════════════════════════════════════════════════════════════════

 1. 💰 COÛT : -2.23€ par unité (-9.7%)
    
 2. 🔋 AUTONOMIE : ×5 à ×13 meilleure selon mode
    
 3. 🔧 SIMPLICITÉ : 
    • 7 composants en moins
    • Circuit compréhensible par débutant
    • Moins de points de défaillance
    
 4. 🛠️ ASSEMBLAGE :
    • LED traversant vs SMD (soudure main facile)
    • Pas besoin de reflow oven
    • Temps assemblage divisé par 2
    
 5. 🌡️ THERMIQUE :
    • Pas de dissipation chaleur significative
    • Pas besoin de via thermiques
    • Température PCB stable
    
 6. 🔬 PERFORMANCE THÉRAPEUTIQUE :
    • CRI >90 conservé (qualité lumière identique)
    • Spectre 5000K conservé
    • Faisceau plus focalisé (15° vs 120°) → meilleur pour stylet!
    
 7. 📦 DISPONIBILITÉ :
    • Stock mondial excellent (composant standard)
    • Multiples fournisseurs
    • Délais courts

 ⚠️ POINTS D'ATTENTION
═══════════════════════════════════════════════════════════════════════════════

 1. FLUX LUMINEUX RÉDUIT :
    • 8lm @ 20mA vs 100lm @ 350mA

    • Suffisant pour auriculothérapie (application de proximité)
    
 2. ORIENTATION LED :
    • LED 5mm = faisceau directionnel
    • Placement précis nécessaire (pointe du stylet)
    • Angle d'incidence critique pour efficacité
    
 3. COURANT LIMITÉ :
    • Max 30mA absolu
    • Pas de mode "boost" possible
    • Adapter protocoles si besoin intensité maximale

projet, avec des bénéfices substantiels :

  • Simplification drastique (circuit divisé par 4)
