══════════════════════════════════════════════════════════════════════════════
  🔧 ASSEMBLAGE FINAL (Post-PCBA)
══════════════════════════════════════════════════════════════════════════════

ÉTAPE 1 : Réception PCB assemblé JLCPCB
  ✅ Vérifier composants SMD présents et orientations correctes
  ✅ Tester continuité +3.3V / GND (multimètre)

ÉTAPE 2 : Soudure composants traversants
  ⚠️ Ordre recommandé :
     1. Connecteur ISP J1
     2. Connecteur batterie J2
     3. 🆕 Connecteur OLED J3 (v8.0)
     4. Boutons SW1, SW2, 🆕 SW3 (v8.0)
     5. LED principale D1 (en dernier, respect polarité)

ÉTAPE 3 : Programmation ATmega328P
  ⚠️ Via ISP (J1) avec USBasp ou Arduino as ISP :
     - Régler fuses (8MHz horloge interne : Low=0xE2, High=0xDA, Ext=0xFD)
     - Uploader bootloader (optionnel)
     - Uploader sketch v8.0 (stylet_auriculo_ATmega328P_v8.0.ino)

ÉTAPE 4 : Connexion OLED
  ⚠️ Vérifier ordre pins câble J3 :
     PCB : GND - VCC - SCL - SDA
     OLED: vérifier datasheet (parfois différent !)
  ⚠️ Test I2C :
     - Scanner adresses I2C (sketch test)
     - Doit détecter 0x20 (PCF8574) et 0x3C (OLED)

ÉTAPE 5 : Installation bibliothèques Arduino
  ⚠️ Installer via Gestionnaire de bibliothèques :
     - Adafruit GFX Library
     - Adafruit SSD1306
     - Wire (incluse par défaut)

ÉTAPE 6 : Tests fonctionnels
  ✅ Boutons : FREQ, MODE, 🆕 MENU fonctionnels
  ✅ LED principale : PWM variable selon mode
  ✅ LEDs RGB : changement couleur selon fréquence
  ✅ OLED : affichage batterie, mode, fréquence
  ✅ Timer : décompte fonctionnel
  ✅ Modulation : oscilloscope si disponible (vérifier 976Hz ± fréquence Nogier)

ÉTAPE 7 : Installation batterie
  ✅ Connecter batterie Li-Ion 3.7V 2000mAh
  ✅ Vérifier polarité J2 (+/-)
  ✅ Mesurer tension batterie sur OLED

ÉTAPE 8 : Boîtier & finitions
  ✅ Impression 3D boîtier (design à créer)
  ✅ Trou pour LED principale D1
  ✅ Accès boutons SW1/SW2/SW3
  ✅ Fenêtre transparente pour OLED
