║  1. ÉCRAN OLED 0.91" I2C - AFFICHAGE PROFESSIONNEL                        ║
╚═══════════════════════════════════════════════════════════════════════════╝

🖥️ **Affichage en temps réel :**
  ✅ Statut ON/OFF + Niveau batterie (%)
  ✅ Mode actif (Détection / Traitement / Full Spectre)
  ✅ Fréquence active (A-G) + valeur en Hz
  ✅ Statut modulation (ON/OFF)
  ✅ Timer/chronomètre de soin

📍 **Position :** Arrière du stylet (câblé via JST-4 pins)
🔌 **Connexion :** I2C partagé avec PCF8574 (adresse 0x3C)
💡 **Auto-extinction :** Après 10s d'inactivité (économie batterie)

╔═══════════════════════════════════════════════════════════════════════════╗
║  2. MODULATION FRÉQUENCES NOGIER RÉIMPLÉMENTÉE                            ║
╚═══════════════════════════════════════════════════════════════════════════╝

🌊 **Principe de modulation :**
  • Porteuse : 976Hz (PWM Timer2)
  • Modulante : 2.28Hz à 146Hz (selon fréquence Nogier sélectionnée)
  • Profondeur : 70% (25% min → 95% max duty cycle)

🎛️ **Contrôle par menu OLED :**
  • Bouton MENU → Accès sous-menu "Modulation"
  • Options : ON / OFF
  • Sauvegarde automatique de la préférence

🧠 **Bénéfices thérapeutiques :**
  ✓ Stimulation pulsée plus douce pour l'œil
  ✓ Meilleure pénétration tissulaire (effet stroboscopique)
  ✓ Confort visuel praticien et patient
  ✓ Respect des fréquences historiques de Nogier

🎛️ **Gestion de la modulation :**

```
Mode DÉTECTION (avec modulation ON) :
  - Duty cycle moyen : 25%
  - Porteuse : 976Hz
  - Modulante : Fréquence Nogier active
  - Profondeur : 70%
  - Formule : duty = 25% ± 35% × sin(2π × freq_nogier × t)

Mode TRAITEMENT (avec modulation ON) :
  - Duty cycle moyen : 92%
  - Porteuse : 976Hz
  - Modulante : Fréquence Nogier active
  - Profondeur : 70%
  - Formule : duty = 92% ± 35% × sin(2π × freq_nogier × t)

Mode BLANC (modulation forcée OFF) :
  - Duty cycle fixe : 50%
  - Pas de modulation
```

⌨️ **Interactions boutons :**

```
FREQ (court) : Cycle fréquences A→B→C→D→E→F→G→A
FREQ (long)  : Affiche détails fréquence sur OLED

MODE (court) : Cycle modes Détection→Traitement→Blanc
MODE (long)  : Affiche détails mode sur OLED

MENU (court) : Ouvre/ferme menu OLED
MENU (long)  : Reset timer si actif
```

 🏗️ MIGRATION v7.0 → v8.0
═══════════════════════════════════════════════════════════════════════════════

✅ **Compatible avec PCB v7.0 actuel ?** OUI (modifications mineures)

**Option A : Modification PCB v7.0 existant**
  1. Ajouter 1 connecteur JST-4 pins sur zone arrière
  2. Câbler : SDA, SCL, VCC, GND vers connecteur
  3. Ajouter bouton MENU sur pin PB0 (ou utiliser fil volant)
  4. Flasher nouveau firmware v8.0

**Option B : Nouveau PCB v7.1 optimisé**
  1. Inclure connecteur JST-4 dans design
  2. Ajouter pad bouton MENU SMD
  3. Marquer "v8.0 ready" sur sérigraphie

═══════════════════════════════════════════════════════════════════════════════
 🎓 GUIDE UTILISATEUR v8.0
═══════════════════════════════════════════════════════════════════════════════

📖 **Démarrage :**
  1. Allumer stylet (bouton ON/OFF)
  2. Écran OLED affiche : "Stylet ON - Freq A"
  3. LEDs RGB indiquent fréquence active

🎛️ **Sélection fréquence :**
  1. Appui court FREQ : Change fréquence (A→G)
  2. OLED affiche : "Freq: B 4.56Hz"
  3. LEDs RGB s'adaptent (codage couleur)

🔄 **Changement mode :**
  1. Appui court MODE : Cycle modes
  2. OLED affiche : "Mode: Traitement"
  3. Intensité LED s'adapte

⚙️ **Activation modulation :**
  1. Appui court MENU : Ouvre menu
  2. Sélectionner "Modulation"
  3. Appui FREQ : Toggle ON/OFF
  4. OLED affiche : "Modulation : ON [MOD]"
  5. LED clignote selon fréquence Nogier

⏱️ **Utilisation timer :**
  1. Dans menu : "Timer : SET"
  2. Appui FREQ : +1 min (1-60)
  3. Appui MODE : START
  4. Compte à rebours sur OLED
  5. Alarme visuelle à 00:00
