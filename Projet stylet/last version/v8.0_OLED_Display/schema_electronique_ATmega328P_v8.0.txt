D1 - LED Nichia NSPW500CS

  🔋 ALIMENTATION & MESURE BATTERIE

J2 - Connecteur Batterie (JST 2.0mm)
  Pin 1 : +VBAT (3.7V nominal)
  Pin 2 : GND

🆕 SW4 - Interrupteur ON/OFF (Coupure batterie) - NOUVEAU v8.0
  Pin 1 (COM)   : J2 Pin 1 (+VBAT brut depuis batterie)
  Pin 2 (NO/ON) : VBAT (réseau alimentation circuit)
  Pin 3 (NC)    : Non connecté
  Type          : Slide switch SPDT (SS-12D00, 7×3mm, traversant)
  Fonction      : Coupure complète alimentation

  Chemin alimentation :
    J2(+) → SW4 COM ──┤ON├── SW4 NO → VBAT → LDO U3 + Diviseur R8/R9
    
    Position OFF : Circuit ouvert, batterie déconnectée → 0µA
    Position ON  : Batterie connectée, circuit alimenté normalement

  ⚠️ AVANTAGE :
    Contrairement au mode veille (sleep ~5µA), l'interrupteur
    garantit 0µA de consommation → aucune décharge batterie

DIVISEUR RÉSISTIF (Mesure tension batterie)
  VBAT ─┬─ R8 (100kΩ) ─┬─ R9 (33kΩ) ─┬─ GND
        │              │              │
        │              └──> PC0 (A0)  │
        │                             │
        └─> VIN (LDO U3 pin 3)        │

  Calcul tension mesurée :
    V_PC0 = VBAT × R9 / (R8 + R9)
    V_PC0 = 4.2V × 33kΩ / 133kΩ = 1.04V  (batterie pleine)
    V_PC0 = 3.0V × 33kΩ / 133kΩ = 0.74V  (batterie vide)
    
    → Plage ADC : 0.74V - 1.04V (bien en dessous 3.3V)
  
  Courant diviseur :
    I = VBAT / (R8 + R9) = 4.2V / 133kΩ = 31.6µA (négligeable)

  🔗 BUS I2C & PULL-UPS
══════════════════════════════════════════════════════════════════════════════

RÉSISTANCES PULL-UP (montage parallèle pour flexibilité) :

R2a - Pull-up SDA (4.7kΩ)
  Pin 1 : SDA (net SDA)
  Pin 2 : +3.3V

R2b - Pull-up SCL (4.7kΩ)
  Pin 1 : SCL (net SCL)
  Pin 2 : +3.3V

⚠️ ALTERNATIVE : Une seule résistance 2.2kΩ sur chaque ligne (équivalent parallèle)

CALCUL RÉSISTANCE PULL-UP :
  R_pullup = (Vcc - Vol) / (Isink_total)
  
  Avec :
    - Vcc = 3.3V
    - Vol = 0.4V (tension LOW I2C standard)
    - Isink = 3mA par périphérique × 2 = 6mA
    
  R_pullup_min = (3.3 - 0.4) / 0.006 = 483Ω  → Sécurité : 2.2kΩ OK
  R_pullup_max = 1000 / (400pF × 100kHz) = 25kΩ  → 4.7kΩ OK
  
  Valeur choisie : 4.7kΩ (standard I2C, fréquence 100kHz)

PÉRIPHÉRIQUES I2C :
  1. PCF8574T  @ 0x20 (U2)
  2. 🆕 OLED SSD1306 @ 0x3C (J3) - NOUVEAU v8.0

AJOUTS SCHÉMATIQUE :
  0. 🆕 Ajouter SW4 (interrupteur ON/OFF) :
     - Symbole : SW_SPDT
     - Footprint : SW_Slide_1P2T_CK_OS102011MA1QN1 (ou SS-12D00)
     - Pin COM → J2 Pin 1 (+VBAT brut)
     - Pin NO  → VBAT (net alimentation circuit)
     - Pin NC  → Non connecté

  1. Ajouter SW3 (bouton MENU) :
     - Symbole : SW_PUSH
     - Footprint : SW_SPST_6x6mm_TH
     - Pin 1 → PB0 (U1 pin 12)
     - Pin 2 → GND
  
  2. Ajouter J3 (connecteur OLED) :
     - Symbole : Conn_01x04
     - Footprint : PinHeader_1x04_P2.54mm_Vertical ou JST_XH_B4B
     - Pin 1 (GND) → GND
     - Pin 2 (VCC) → +3.3V
     - Pin 3 (SCL) → SCL (net existant)
     - Pin 4 (SDA) → SDA (net existant)
  
  3. Modifier flag "NC" sur PB0 :
     - U1 pin 12 : SUPPRIMER le flag "No Connect"
     - Connecter à SW3

VÉRIFICATIONS ERC :
  ✅ Pas de nouveau "Pin not connected" (PB0 maintenant utilisé)
  ✅ SDA/SCL bien partagés entre 3 composants (U1, U2, J3)
  ✅ Pull-ups toujours présents (R2a, R2b)

AJOUTS PCB :
  0. 🆕 Placer SW4 (interrupteur ON/OFF) :
     - Position : bord latéral PCB (accessible depuis tube)
     - Face : B.Cu (arrière)
     - Orientation : levier vers bord PCB

  1. Placer SW3 :
     - Position : entre SW1 et SW2 (alignement)
     - Face : B.Cu (arrière)
  
  2. Placer J3 :
     - Position : bord supérieur PCB (côté LED principale)
     - Orientation : pins vers extérieur
     - Face : F.Cu (avant) ou B.Cu (selon préférence)
  
  3. Router nouvelles pistes :
     - PB0 → SW3
     - J3 pins → nets existants (GND, +3.3V, SDA, SCL)

VÉRIFICATIONS DRC :
  ✅ Clearances respectées
  ✅ Pas de pistes non connectées
  ✅ Zones GND mises à jour (vias ajoutés si besoin)
